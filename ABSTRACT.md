## Title
marinehabitat3d: An R Package for Three-Dimensional Marine Spatial Analyses

## Authors and Affiliations
Jay H. Matsushiba, Nicholas K. Dulvy, Rachel Aitchison

## Abstract

The three-dimensional marine environment in which sharks, rays, and chimaeras live poses a unique challenge for spatial analyses. Most conventional Geographic Information System (GIS) workflows represent space as a two-dimensional plane. This abstraction fails to accurately represent the range of depths and vertical overlap of various marine habitat types. Additionally, fisheries operate at different depths depending on factors like gear type, creating depth-specific patterns of threat exposure that require volumetric analyses to quantify. These reasons demonstrate the need for a reproducible and easy-to-use method for querying 3D space for chondrichthyan species ranges and distributions. In this presentation, we share progress on the development of marinehabitat3d, an open-source R package designed to facilitate three-dimensional marine spatial analyses. The package provides functions for extracting values from depth-stratified oceanographic rasters within user-defined geographic areas and depth ranges, enabling researchers to work with any gridded marine dataset. This package will also incorporate tools for calculating three-dimensional overlap between species distributions and fisheries effort. We demonstrate the package using an example workflow that integrates IUCN Red List species ranges with World Ocean Atlas 2023 climatologies, extracting temperature and dissolved oxygen profiles for 1,254 chondrichthyan species. The marinehabitat3d package provides tools and documented workflows for characterizing shark, ray, and chimaera habitat. By making three-dimensional marine analyses accessible and reproducible, marinehabitat3d can accelerate research across all sharks, rays, and chimaeras—from habitat characterization to climate vulnerability to fisheries risk assessment.
